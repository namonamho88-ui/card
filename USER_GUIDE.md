# Cherry Picker Agent - 사용자 가이드

## 🚀 시작하기

### 설치 및 실행

#### 1. 프로젝트 클론
```bash
git clone https://github.com/namonamho88-ui/card.git
cd card
```

#### 2. 의존성 설치
```bash
npm install
```

#### 3. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

#### 4. 프로덕션 빌드
```bash
npm run build
npm run preview
```

## 📖 주요 기능 사용법

### 1. 카드 탐색하기

#### 카드사별 필터링
1. 페이지 상단의 **"🏆 카드사별 인기 TOP10 카드 목록"** 섹션으로 이동
2. 7개 탭 중 하나를 선택:
   - **전체**: 60종 모든 카드 표시
   - **신한카드**: 신한카드 10종만 표시
   - **현대카드**: 현대카드 10종만 표시
   - **삼성카드**: 삼성카드 10종만 표시
   - **우리카드**: 우리카드 10종만 표시
   - **하나카드**: 하나카드 10종만 표시
   - **롯데카드**: 롯데카드 10종만 표시

#### 카드 그리드 보기
- 카드는 그리드 형태로 표시됩니다
- 각 카드에는 다음 정보가 표시됩니다:
  - 카드사명 (상단)
  - 카드명 (중앙, 큰 글씨)
  - 연회비 (하단)

#### 카드 호버 효과
- 마우스를 카드 위에 올리면:
  - 카드가 위로 살짝 올라옵니다
  - 그림자가 강해집니다
  - 그라디언트 오버레이가 나타납니다

---

### 2. 카드 상세 정보 확인

#### 카드 클릭
1. 원하는 카드를 클릭합니다
2. 상세 정보 모달이 나타납니다

#### 모달에서 확인 가능한 정보
- **카드 프리뷰**: 카드사 브랜드 컬러로 표시된 카드 이미지
- **카드 정보**:
  - 💳 연회비
  - 📊 전월 실적 조건
- **주요 혜택**: 3가지 핵심 혜택
- **카드 신청하기 버튼**: 클릭 시 알림 표시

#### 모달 닫기
- ✕ 버튼 클릭
- 모달 외부 영역 클릭
- ESC 키 누르기 (브라우저 기본 동작)

---

### 3. AI 카드 추천 받기

#### 챗봇 사용하기
1. 페이지 하단의 **"🤖 AI 카드 추천"** 섹션으로 이동
2. 입력창에 질문을 입력합니다
3. Enter 키를 누르거나 "전송" 버튼을 클릭합니다

#### 효과적인 질문 방법

**✅ 좋은 질문 예시:**
```
"영화를 자주 보는데 제일 혜택 좋은 카드는?"
"스타벅스 할인 카드 추천해줘"
"주유 할인이 많은 카드 알려줘"
"대중교통 할인 카드 찾고 있어"
"온라인 쇼핑 많이 하는데 어떤 카드가 좋아?"
"배달 음식 자주 시키는데 추천 카드는?"
"해외 여행 갈 때 쓸 카드 추천"
"마트에서 쓸 카드 찾고 있어"
```

**❌ 피해야 할 질문:**
```
"좋은 카드 추천해줘" (너무 광범위)
"카드 알려줘" (구체적이지 않음)
"이거 뭐야?" (맥락 없음)
```

#### AI 응답 이해하기

**응답 형식:**
```
**[카드사] [카드명]** 카드를 추천드립니다!

💳 **연회비**: 15,000원
📊 **전월 실적**: 30만원

✨ **주요 혜택**:
1. 스타벅스 30% 할인
2. 카페 전문점 20% 할인
3. 베이커리 15% 할인

📋 **다른 추천 카드**:
2. 현대카드 Green (연회비: 15,000원)
3. 삼성카드 taptap CAFE (연회비: 18,000원)
4. 우리카드 위비 카페 (연회비: 15,000원)
```

**응답 해석:**
- **1순위 카드**: 가장 혜택이 좋은 카드 (상세 정보 포함)
- **다른 추천 카드**: 2~4순위 카드 (간략 정보)
- **점수 기준**: 키워드 매칭 + 할인율 + 카테고리 일치도

## 💡 활용 팁

### 카드 선택 시 고려사항

#### 1. 연회비 vs 혜택
```
연회비가 낮다고 무조건 좋은 것은 아닙니다!

예시:
- A카드: 연회비 10,000원, 스타벅스 20% 할인
- B카드: 연회비 25,000원, 스타벅스 30% 할인

월 스타벅스 지출이 10만원이라면?
- A카드: 연간 혜택 24만원 - 연회비 1만원 = 23만원 이득
- B카드: 연간 혜택 36만원 - 연회비 2.5만원 = 33.5만원 이득
→ B카드가 더 유리!
```

#### 2. 전월 실적 확인
```
전월 실적을 채우지 못하면 혜택을 받을 수 없습니다!

예시:
- 카드: 전월 실적 30만원 필요
- 월 평균 지출: 20만원
→ 실적 미달로 혜택 못 받음
→ 실적 조건이 낮은 다른 카드 고려
```

#### 3. 혜택 카테고리 집중
```
자주 사용하는 카테고리에 집중하세요!

예시:
- 영화 월 2회 → 영화 전용 카드
- 스타벅스 주 3회 → 카페 전용 카드
- 주유 주 1회 → 주유 전용 카드
```

### 카드사별 특징

#### 신한카드
- **특징**: 포인트 적립형 강점
- **추천**: 일상 소비가 다양한 사용자
- **대표 카드**: 딥 드림 (0.7~3.5% 적립)

#### 현대카드
- **특징**: 디자인 우수, M포인트 시스템
- **추천**: 포인트 유효기간 신경 쓰기 싫은 사용자
- **대표 카드**: M (무료, 포인트 무기한)

#### 삼성카드
- **특징**: 디지털 혜택 강점
- **추천**: 온라인 쇼핑, 배달 많이 하는 사용자
- **대표 카드**: iD (온라인 7%, 배달 10%)

#### 우리카드
- **특징**: 단순 할인형
- **추천**: 복잡한 조건 싫어하는 사용자
- **대표 카드**: 에브리원 (모든 곳 1% 할인)

#### 하나카드
- **특징**: 복합 혜택형
- **추천**: 여러 카테고리 골고루 사용하는 사용자
- **대표 카드**: Viva (주유+편의점+교통)

#### 롯데카드
- **특징**: 롯데 계열사 강점
- **추천**: 롯데마트, 롯데시네마 자주 가는 사용자
- **대표 카드**: Cinema (롯데시네마 50%, 월 5회)

## 📱 모바일 사용 가이드

### 터치 제스처
- **탭**: 카드 선택, 탭 전환
- **스크롤**: 카드 그리드 탐색
- **핀치 줌**: 지원 안 함 (브라우저 기본 동작)

### 모바일 최적화 기능
- **2열 그리드**: 600px 이하 화면
- **1열 그리드**: 400px 이하 화면
- **큰 터치 영역**: 탭 버튼, 카드 아이템
- **읽기 쉬운 폰트**: Noto Sans KR

### 모바일 팁
1. **가로 모드**: 더 많은 카드를 한 번에 볼 수 있습니다
2. **탭 스크롤**: 탭이 많을 경우 좌우 스크롤 가능
3. **모달 스크롤**: 혜택이 많을 경우 모달 내부 스크롤

## 🔍 FAQ

### Q1. 카드 데이터는 실시간인가요?
**A**: 아니요, 정적 데이터입니다. 실제 카드사 홈페이지에서 최신 정보를 확인하세요.

### Q2. 카드 신청이 가능한가요?
**A**: 아니요, "카드 신청하기" 버튼은 알림만 표시합니다. 실제 신청은 카드사 홈페이지에서 하세요.

### Q3. 내 카드를 추가할 수 있나요?
**A**: 현재 버전에서는 불가능합니다. 60종 고정 카드만 제공됩니다.

### Q4. 오프라인에서 사용 가능한가요?
**A**: 아니요, 인터넷 연결이 필요합니다.

### Q5. AI 추천이 정확한가요?
**A**: 키워드 기반 매칭이므로 참고용으로만 사용하세요. 실제 선택은 본인의 소비 패턴을 고려하세요.

### Q6. 여러 카드를 비교할 수 있나요?
**A**: 현재 버전에서는 불가능합니다. 한 번에 한 카드씩만 확인 가능합니다.

### Q7. 즐겨찾기 기능이 있나요?
**A**: 아니요, 현재 버전에서는 지원하지 않습니다.

### Q8. 모바일 앱이 있나요?
**A**: 아니요, 웹 애플리케이션만 제공됩니다. 모바일 브라우저에서 사용하세요.

## 🐛 문제 해결

### 카드가 표시되지 않아요
1. 브라우저를 새로고침하세요 (Ctrl+F5 또는 Cmd+Shift+R)
2. 브라우저 캐시를 삭제하세요
3. 다른 브라우저에서 시도하세요

### 탭이 작동하지 않아요
1. JavaScript가 활성화되어 있는지 확인하세요
2. 브라우저를 업데이트하세요
3. 콘솔 에러를 확인하세요 (F12)

### 모달이 닫히지 않아요
1. ✕ 버튼을 클릭하세요
2. 모달 외부를 클릭하세요
3. ESC 키를 누르세요
4. 페이지를 새로고침하세요

### AI 챗봇이 응답하지 않아요
1. 질문이 너무 짧지 않은지 확인하세요
2. 구체적인 키워드를 포함하세요 (예: 영화, 카페, 주유)
3. 600ms 후 응답이 나타나므로 잠시 기다리세요

### 모바일에서 레이아웃이 깨져요
1. 브라우저를 최신 버전으로 업데이트하세요
2. 화면을 회전해보세요 (가로/세로)
3. 줌을 100%로 설정하세요
